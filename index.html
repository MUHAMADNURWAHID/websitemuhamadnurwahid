Tentang Saya

Saya adalah mahasiswa Teknik Informatika Universitas Pamulang yang memiliki ketertarikan pada pengembangan web dan teknologi digital. Selain mempelajari dasar-dasar pemrograman, saya juga aktif mengembangkan proyek kecil sebagai latihan untuk meningkatkan kemampuan teknis saya. Saya memiliki minat pada kegiatan olahraga seperti sepak bola dan mendaki gunung. Saat ini saya fokus memperdalam HTML, CSS, dan JavaScript sebagai langkah awal untuk menjadi developer yang kompeten dan profesional.
